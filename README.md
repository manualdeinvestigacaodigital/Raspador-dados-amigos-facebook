\# Raspador de dados de amigos do Facebook



Extensão Chrome para raspagem de dados relacionais do Facebook, com coleta online em tempo real, enriquecimento de Bio/Metadados e exportações HTML, JSON, CSV e XLSX.



\## Instalação



1\. Acesse `chrome://extensions/`

2\. Ative o modo de desenvolvedor

3\. Clique em `Carregar sem compactação`

4\. Selecione a pasta:



`EXTENSAO`



Ou, no Windows:



`D:\\Programação\\Roseta\\novo\\Raspador de dados de amigos do Facebook\\EXTENSAO`



\## Arquivos da extensão



\- `EXTENSAO/manifest.json`

\- `EXTENSAO/content.js`

\- `EXTENSAO/main\_world\_hook.js`

\- `EXTENSAO/service\_worker.js`
